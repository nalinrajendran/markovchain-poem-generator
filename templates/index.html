Poem Generator

TEAM-13

This website is a Markov chain-based poetry generator chain that simulates poetic line breaks by including a new line token in its model. It also includes the option of introducing an element of randomness, where by each step in walking through the chain has a chance of moving to any other state before continuing.

MARKOV CHAINS


A Markov chain is a stochastic model describing a sequence of possible events in which the probability of each event depends only on the state attained in the previous event.

A Markov chain can be viewed as a finite state automata, with words representing states, and each edge containing a probability of transition information.

IMPLEMENTATION


For each word a list of possible followups is main- tained. This list is built in a learning part, while it- erating through poems. Important thing to note is that duplicates are not dropped. This captures the Markov process: more frequent word in a list has higher probability of being chosen as a followup.

First word of the new generated poem is chosen randomly. Next subsequent word is chosen from the according list (built during learning) randomly, adjusted for the fact that more frequent words have higher probability of being chosen. The process repeats until the desired length is reached. Length is defined by number of lines and number of words per line.
